# Integrate React into Drupal
A working example.

## Quick start
1. Copy `.example.env` into `.env` 
2. Run `ahoy up` 
3. Run `ahoy composer install` 
4. Run `ahoy prepare` 
4. Run `ahoy install` 
5. Visit http://localhost:8080/web

Login: `admin` / `admin` 
### Environment
- Web server: localhost:8080
- PHPMy Admin: localhost:8090
- MySQL: 
    - localhost:3306 || mysql:3306
    - u: `root` p: `none` 

### Shell access
- `ahoy sh`
